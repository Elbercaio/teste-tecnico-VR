# teste-tecnico-VR

Para rodar o projeto

# Clone

```sh
git clone git@github.com:Elbercaio/teste-tecnico-VR.git
cd teste-tecnico-VR
```

# Crie um arquivo .env em /api

```sh
cd api
cp .env.example .env
cd ..
```

# Suba os conteiners

```sh
docker compose up -d
```

# Acesse

```sh
http://localhost/
```
